# SkillPath AI

## AI-Powered Personalized Learning Platform

SkillPath AI is a full-stack MERN application that generates personalized learning roadmaps using Artificial Intelligence. It helps learners follow a structured learning path based on their learning goals, current skill level, and weekly study commitment. The project is aligned with **United Nations Sustainable Development Goal (SDG) 4 – Quality Education**.

## Problem Statement

Many learners struggle to identify a structured learning path because educational resources are scattered across multiple platforms. This often leads to confusion, inconsistent learning, and difficulty tracking progress.

## Solution

SkillPath AI addresses this challenge by generating personalized learning roadmaps tailored to each learner. It also provides AI-powered learning assistance, topic-wise resources, quizzes, and progress tracking within a single platform.

## Key Features

- Secure User Authentication (JWT)
- Personalized AI Learning Roadmaps
- Learning Goal Selection
- Beginner, Intermediate & Advanced Levels
- Weekly Study Time Planning
- AI Chatbot for Learning Assistance
- Topic-wise Learning Resources
- Topic-wise Quizzes
- Progress Tracking Dashboard
- Responsive User Interface
- Light & Dark Theme Support
